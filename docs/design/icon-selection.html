DesktopCleaner 图标方案

当前正式采用 C「桌面入口」方向：深色面板轮廓、入口方块、粉色与青色强调。主应用和托盘图标都基于同一套识别语言。

已选择：C

C 桌面入口

主应用图标。识别度比显示器图标高，适合 exe、任务栏、窗口标题栏和快捷方式。

C 托盘简化版

托盘图标。减少细节，保留面板轮廓和两个色块，避免 16px 下糊成一团。

接入状态

正式资产位于 assets/icons/app.ico 和 assets/icons/tray.ico。构建脚本已经使用主图标。

验收点

资源管理器、任务栏、设置窗口左上角应显示主应用图标；系统托盘应显示简化图标。普通文件、文件夹和快捷方式继续使用 Windows 原生图标。
